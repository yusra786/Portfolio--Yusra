# Yusra Waseem Portfolio

A modern, responsive personal portfolio built with React and TailwindCSS.

## Features
- Dark modern UI with neon accents (Purple/Blue).
- Glassmorphism effects for cards and components.
- Smooth scrolling navigation.
- Comprehensive sections: Hero, Skills, Projects, Timeline (Training, Education, Certifications, Participation), and Contact.
- Responsive design tailored for mobile and desktop viewing.
